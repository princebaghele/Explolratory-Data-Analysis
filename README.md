This project applies machine learning techniques to predict the age of abalone based on physical measurements. Using the Abalone dataset, the workflow includes data preprocessing, exploratory data analysis, feature engineering, and model building.

Multiple regression models such as Linear Regression, Ridge Regression, Decision Tree, and Random Forest were trained and evaluated using performance metrics like Mean Squared Error (MSE) and R² score. The goal is to identify the most accurate model for predicting abalone age (measured in rings) from given features.
